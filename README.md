# Student Manager

###### Java Assignment 2 - Zecheriah Ferguson



### Project Description

This is a text-based student manager. You can freely create, delete, search for and display all student information.



###### Students

When creating a new student entry (menu option 1), you must include their name, email, student ID and courses. They will then be added to a list of students. Two sample students, Zech and Aiden, are already pre-registered, and you can view them by selecting menu option 4 ("display students and instructors").



###### Instructors

Instructors are also displayed, but cannot be created or edited. Reason being, the assignment instructions included adding an Instructor class but never said what to do with it, so my interpretation of it was to hardcode some instructors in and simply display them with the students.



### Setup Instructions

NOTE: You need Java installed in order to run the program.



1. Open the entire folder within Visual Studio Code.
2. Select "StudentManager.java"
3. At the top right of the screen, click the "Run Java" play button. A console will open and you will be brought to the main menu.
